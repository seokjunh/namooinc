## 프로젝트 설치
```bash
git clone https://github.com/seokjunh/namooinc.git
```

## Nextjs

### 1. 의존성 설치

```bash
cd namooinc/front-nextjs
npm install
```

### 2. 실행

```bash
npm run dev
```

## Springboot + sql server
